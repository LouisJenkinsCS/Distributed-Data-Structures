class IndexableList: Collection¶

A list that can be indexed into.

proc get(idx: int): (bool, eltType)¶

Obtain the element at the requested index.

proc indexOf(elt: eltType): int¶

Obtains the index of the requested element, if present in the list.

proc add(idx: int, elt: eltType): bool¶

Add an element at a specific index in the list.

proc subList(start: int, end: int = -1): IndexableList(eltType)¶

Creates a new list containing the items at the specified indexes. If end is less than start, then the end indice is set to the end of the list. If the end is greater than the size of the list, it will also be set to the end of the list.
